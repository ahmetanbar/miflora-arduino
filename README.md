# Info

This project is for using Xiaomi MiFlora BLE Sensor by Arduino. Xiaomi has BLE, Bluetooth Low Energy. It is Bluetooth 4.0. So, we use HM-10 sensor able to contact miflora. It supports Bluetooth 4.0.

Most probably, your HM-10 sensor will need upgrade. You need to follow this [link](http://martyncurrey.com/hm-10-bluetooth-4ble-modules/).

## Installation

Clone the project, and compile the `miflora-arduino.ino` file and upload to your Arduino.

## Usage

VCC -> 3.3V

GND -> GND

RX  -> TX

TX  -> RX
